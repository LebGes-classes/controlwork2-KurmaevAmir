1 Что такое наследование в Java и как оно используется?
Наследование в Java - использование свойств и методов родительского класса.
Реализуется расширением наследуемого класса через extends.
К примеру есть класс Human и Student. В данном случае, не у каждого Human есть права Student. Для реализации в классе Students будет реализован метод extends Human.
Так как любой студент - человек, но не любой человек - студент.
2. Какие преимущества и недостатки имеет множественное наследование в контексте Java?
В контексте java множественное наследование недоступно для обычных классов, зато может использоваться в интерфейсах.
Преимущества:
Уменьшение копирования кода;
Появление чёткой иерархии;
Недостатки:
Правило ромба;
Усложнение читаемости кода;
3. Что такое полиморфизм в объектно-ориентированном программировании?
Под понятием полиморфизма понимается возможность обработки различных типов одинаково. Пример использование - интерфейс.
4. Как реализуются полиморфизм в Java и использованием методов?
В Java полиморфизм реализуется через наследование, интерфейсы. При использовании полиморфизма в Java, код будет работать с различными типами объектов.
5. Какие виды полиморфизма существуют в Java?
Перезагрузка методов и операций; наследование и переопределение методов.
